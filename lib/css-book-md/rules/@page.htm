语法：

@page <label> <pseudo-classes>{ sRules }

取值：

<label>：
页面标识符
<pseudo-class>：
打印伪类[:first](../selectors/pseudo-classes/@page-first.htm), [:left](../selectors/pseudo-classes/@page-first.htm), [:right](../selectors/pseudo-classes/@page-first.htm)

说明：

设置页面容器的版式，方向，边空等。

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
IE Firefox Chrome Safari Opera iOS Safari Android Browser Android Chrome
6.0-7.0 2.0-18.0 4.0+ 3.1+ 15.0+ 3.2+ 2.1+ 18.0+
9.0+ 19.0+
